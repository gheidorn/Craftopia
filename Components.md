# Components

## Processing Tools & Materials

### Tools
- **Workbench** - Basic crafting station
- **Master Workbench** - Advanced crafting station
- **Saw** - Basic wood cutting tool
- **Fine Saw** - Improved wood cutting tool
- **Master Saw** - Expert wood cutting tool
- **Furnace** - Basic smelting station
- **Forge** - Advanced smelting station
- **Master Forge** - Expert smelting station
- **Cutting Board** - Basic food preparation surface
- **Master Cutting Board** - Advanced food preparation surface
- **Knife** - Basic cutting tool
- **Sharp Knife** - Improved cutting tool
- **Master Knife** - Expert cutting tool
- **Mortar and Pestle** - Basic grinding tool
- **Alchemy Kit** - Advanced alchemy station
- **Master Alchemy Kit** - Expert alchemy station
- **Spinning Wheel** - Basic rope making tool
- **Master Spinning Wheel** - Advanced rope making tool

### Consumables & Materials
- **Coal** - Fuel for smelting
- **Flux** - Smelting additive
- **Quenching Oil** - Metal treatment
- **Sandpaper** - Wood finishing material
- **Enchanted Oil** - Magical wood treatment
- **Ice** - Food preservation
- **Preservative Salt** - Food preservation
- **Water** - Basic liquid
- **Purified Water** - Cleaned liquid
- **Distilled Water** - Highly purified liquid
- **Alcohol** - Alchemical solvent
- **High-Grade Alcohol** - Premium alchemical solvent
- **Glass Vial** - Storage container
- **Wax** - Rope treatment
- **Resin** - Binding agent
- **Metal Wire** - Reinforcement material
- **Enchanted Thread** - Magical binding material
- **Arcane Crystal** - Magical catalyst
- **Enchanted Catalyst** - Magical alchemical catalyst
